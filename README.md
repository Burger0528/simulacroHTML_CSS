## 1. Description (EN)

This project is a responsive landing page created using only HTML and CSS. It focuses on layout structure, responsive design, and modern CSS techniques such as Grid, Flexbox, media queries, and CSS-only interactions.

The project includes a responsive navigation bar with a hamburger menu for mobile devices, multiple content sections, and a clean visual hierarchy. No JavaScript or external frameworks were used.

## Descripción (ES)

Este proyecto es una landing page responsiva creada únicamente con HTML y CSS. Se enfoca en la estructura del layout, el diseño responsivo y el uso de técnicas modernas de CSS como Grid, Flexbox, media queries e interacciones solo con CSS.

El proyecto incluye una barra de navegación responsiva con menú hamburguesa para dispositivos móviles, múltiples secciones de contenido y una jerarquía visual clara. No se utilizó JavaScript ni frameworks externos.
